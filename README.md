# WLKOM — Wild Linux Kernel Object Module

## Description
Pedagogical Linux rootkit as a kernel module + C2 attacking program.

## Build

### Rootkit
```bash
./setup_vms.sh
```

En attendant, lancer les makefiles :
```
make -C ./rootkit
make -C ./attacking_program
```

Puis suivre les instructions du script lorsqu'il se finit.
